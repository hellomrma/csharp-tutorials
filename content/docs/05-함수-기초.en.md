# 5. Function Basics

## What is a Function?

A function is a block of code that performs a specific task. Instead of repeating the same code multiple times, you can create a function and reuse it.

## Creating a Function

```csharp
void EnemyAttack()
{
    Debug.Log("Enemy Attack!");
    Debug.Log("Enemy Damaged");
    Debug.Log("Enemy HP UI Update");
}
```

- **Components**
  - `void`: means this function doesn't return a value
  - `EnemyAttack`: function name (can be anything you want)
  - `{ }`: contains the code to execute

## Calling a Function

```csharp
void Start()
{
    EnemyAttack();  // Execute function
    EnemyAttack();  // Execute again
    EnemyAttack();  // Execute again
}
```

- **Advantages**
  - No need to write the same code multiple times
  - Modify in one place when needed
  - Code is clean and easy to read

## Why Use Functions?

**Without functions:**
```csharp
Debug.Log("Enemy Attack!");
Debug.Log("Enemy Damaged");
Debug.Log("Enemy HP UI Update");

Debug.Log("Enemy Attack!");
Debug.Log("Enemy Damaged");
Debug.Log("Enemy HP UI Update");

Debug.Log("Enemy Attack!");
Debug.Log("Enemy Damaged");
Debug.Log("Enemy HP UI Update");
```

**With functions:**
```csharp
EnemyAttack();
EnemyAttack();
EnemyAttack();
```

Much simpler and easier to manage!
