# 6. Functions with Return Values

## return Keyword

A way to return calculated results from a function.

```csharp
int ThisYear()
{
    return 2025;  // Return 2025
}
```

- **Usage**
```csharp
void Start()
{
    int year = ThisYear();  // year stores 2025
    Debug.Log(year);        // Outputs 2025
}
```

## Return Types

When creating a function, you must specify what type of value it returns.

```csharp
int ThisYear()      // Returns int type
{
    return 2025;
}

string GetName()    // Returns string type
{
    return "John";
}

bool IsAlive()      // Returns bool type
{
    return true;
}
```

## Using Variables Inside Functions

You can create and use variables inside functions.

```csharp
int NextYear()
{
    int lastYear = 2025;      // Declare variable inside function
    int nextYear = lastYear + 1;  // Use variable
    return nextYear;          // Return result
}
```

- **Variable Scope**
  - Variables created inside a function can only be used within that function
  - Cannot be used in other functions
