# 7. Functions with Parameters

## What are Parameters?

A way to pass values to functions for use. Similar to entering numbers into a calculator.

```csharp
int NextYear(int thisYear)  // Receive and use thisYear
{
    int nextYear = thisYear + 1;
    return nextYear;
}
```

## Passing Values When Calling Functions

```csharp
void Start()
{
    int result = NextYear(2025);  // Pass 2025
    Debug.Log(result);             // Outputs 2026
}
```

- **Process**
  1. Call `NextYear(2025)`
  2. `thisYear` becomes 2025 inside function
  3. Calculate `thisYear + 1` → 2026
  4. Return 2026

## Advantages of Parameters

**Without parameters (inefficient):**
```csharp
int NextYear2025() { return 2026; }
int NextYear2026() { return 2027; }
int NextYear2027() { return 2028; }
// ... must keep creating more
```

**With parameters (efficient):**
```csharp
int NextYear(int thisYear) 
{ 
    return thisYear + 1; 
}

// Can use for any year!
NextYear(2025);  // 2026
NextYear(2026);  // 2027
NextYear(2050);  // 2051
```
