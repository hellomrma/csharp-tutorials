# 8. Using Multiple Parameters

## Passing Multiple Parameters

You can pass multiple values to a function.

```csharp
int PlusNumber(int a, int b)  // Two parameters
{
    int c = a + b;
    return c;
}
```

## Usage Example

```csharp
void Start()
{
    int result = PlusNumber(3, 97);  // Pass 3 and 97
    Debug.Log(result);                // Outputs 100
}
```

- **Process**
  1. Call `PlusNumber(3, 97)`
  2. Set `a = 3`, `b = 97`
  3. Calculate `c = 3 + 97` → 100
  4. Return 100

## Practical Usage

```csharp
// Calculate distance between two points
float Distance(float x1, float y1, float x2, float y2)
{
    float dx = x2 - x1;
    float dy = y2 - y1;
    return Mathf.Sqrt(dx * dx + dy * dy);
}

// Display player information
void ShowPlayerInfo(string name, int level, int hp)
{
    Debug.Log("Name: " + name);
    Debug.Log("Level: " + level);
    Debug.Log("HP: " + hp);
}
```
