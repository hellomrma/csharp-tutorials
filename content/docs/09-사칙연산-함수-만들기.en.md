# 9. Creating Arithmetic Functions

## Addition, Subtraction, Multiplication, Division Functions

```csharp
int PlusNumber(int a, int b)
{
    return a + b;  // Add
}

int MinusNumber(int a, int b)
{
    return a - b;  // Subtract
}

int MultiplyNumber(int a, int b)
{
    return a * b;  // Multiply
}

int DivideNumber(int a, int b)
{
    return a / b;  // Divide
}
```

## Usage Example

```csharp
void Start()
{
    Debug.Log(PlusNumber(3, 97));      // 100
    Debug.Log(MinusNumber(97, 3));     // 94
    Debug.Log(MultiplyNumber(97, 3));  // 291
    Debug.Log(DivideNumber(97, 3));    // 32 (integer division)
}
```

## Function Naming Tips

Function names should clearly indicate what they do.

- ✅ Good examples: `PlusNumber`, `CalculateDamage`, `GetPlayerHP`
- ❌ Bad examples: `func1`, `test`, `abc`

## Function Writing Checklist

1. **Function name**: Does it clearly express the functionality?
2. **Parameters**: Does it receive all necessary values?
3. **Return value**: Does it return the result properly?
4. **Reusability**: Can it be used in other places?
